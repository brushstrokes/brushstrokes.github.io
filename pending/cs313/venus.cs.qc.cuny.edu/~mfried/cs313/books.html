Books

All of the material that you need to know for this course is explained in my notes, so the books are optional.
On the main page you'll find a list of readings for each topic. The reading that I think is most useful is listed first. I do not recommend reading everything, because there's a lot of overlap, but if the first reading confuses you then read the others.

Algorithms (4th ed.) by Robert Sedgewick, Kevin Wayne, 2011
If you only get one book, I recommend this one. It has clear explanations and Java code examples. However, some of the course topics are not covered.

Data Structures & Algorithms in Java (6th ed.) by Goodrich, Tamassia, Goldwasser, 2014
This book covers almost all of the course topics, with Java code examples. It's the only book on this list that covers AVL trees.

The Algorithm Design Manual (2nd ed.) by Steven S. Skiena, 2010
This book focuses more on algorithm design, with good practical advice on how to think about problem solving. It has code examples in both C and pseudocode.

Introduction to Algorithms (3rd ed.) by Cormen, Leiserson, Rivest, Stein, 2009
This book is more mathematical than the others, with more focus on analysis and proofs. The code examples are pseudocode.
