Lists

A list is an ordered sequence of values that allows us to read or modify values at specific positions, and add or remove elements.
It can be implemented with either an array or a linked list.

Arrays

An array is a fixed-size collection of elements of the same type stored continuously in memory.
You can access an element by its index in constant time.
An array cannot grow.
ArrayList is an array implementation of the List interface.

Linked lists


Linked list. Source: geeksforgeeks.org
A linked list is a collection of nodes. Each node contains an element of the list.
In a singly-linked list, each node has a pointer to the next node.
In a doubly-linked list, each node has a pointer to the next node and the previous node.

The nodes are not stored continuously in memory, and a linked list can grow.
We can traverse the list by starting at the first node (the head) and following the pointers from each node to the next node.
LinkedList (JCF) is a doubly-linked list implementation of the List interface.
