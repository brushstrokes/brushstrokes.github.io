Final advice

The final mainly covers material from the second half of the semester, but you still need to know some things from the first half, for example big O notation and how to use lists. There will be no questions about AVL trees.

Some important topics for the final are listed below. The percentages are estimates and may not be accurate.
  • Know how to write Java code (30-40%).
    • You will have to write a sort method (mergesort or quicksort), a tree traversal method (in-order or level-order), and a method that uses sets or maps.
  • Know the runtimes of each data structure operation and algorithm (10-20%).
  • Know how to add and remove elements from a heap, and how to add elements to both types of hash table (10-15%).
  • Know how graph traversal and Dijkstra's algorithm work (10-15%).
  • Be prepared to answer various other questions (most will be similar to the practice problems).
