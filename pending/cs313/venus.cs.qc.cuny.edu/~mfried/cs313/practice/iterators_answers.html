Questions to talk about in class

Suppose you have a linked list of integers. How would you remove all negative integers? What's the runtime?
Use an iterator. Use the iterator's remove method. Do not use the linked list's get method because then it would take Θ(n) to access each element.

Many data structures inherit a toString method from the AbstractCollection class.
How would you write one toString method in AbstractCollection that works for all of the data structures?
Use an iterator. All collections implement the Iterable interface, so they all have an iterator method.
Append each element to a String (or StringBuilder for better efficiency).
