Dynamic programming problems

Write a method
int minPath(int[][] grid)
that returns the length of the shortest path from top-left to bottom-right of the grid.
The "length" is the sum of the numbers along the path.
You can only move in two directions, down or right.

Example 1 3 1 The shortest path is 1+3+1+1+1, so return 7 1 5 1 4 2 1

Write a method
int minCoins(int[] coins, int total)
that returns the fewest coins required to add up to total (or -1 if it is impossible).
The coins array contains the values of each possible coin. You have an infinite supply of each coin.

Examples coins = {1, 5, 8} total = 10 return 2 (5 + 5 = 10) coins = {3, 8} total = 10 return -1 (impossible)
