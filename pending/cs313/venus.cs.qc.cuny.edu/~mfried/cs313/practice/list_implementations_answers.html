What is the worst-case runtime of each operation? Give your answer in Θ notation as a function of n (the number of elements).
ArrayList
Accessing an element by index
Θ(1).
Inserting a new element at the beginning (at index 0)
Θ(n) because you have to shift all the elements to the right.
LinkedList
Accessing an element by index
Θ(n) because you have to traverse the list.
Inserting a new element by index
Θ(n) because you have to traverse the list to reach the point of insertion.
