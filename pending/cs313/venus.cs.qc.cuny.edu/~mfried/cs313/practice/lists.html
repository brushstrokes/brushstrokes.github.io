List<Integer> even(List<Integer> input)

Write a method
int firstDuplicate(int[] a)
that returns the first duplicate in array a. If there are no duplicates, return -1. You may assume that the value of every element is in the range of 0 to n-1 where n is the array length. The runtime must be O(n).
