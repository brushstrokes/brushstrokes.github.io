Graphs questions

How much space does an array of adjacency lists use? Give your answer in Θ notation as a function of V (the number of vertices) and E (the number of edges).
Θ(V+E)

Multiple choice
If a graph has 10000 nodes and 50000 edges, and you want to use as little space as possible, how should you represent it?
A. Adjacency matrix
B. Array of adjacency lists
Array of adjacency lists. The space is Θ(V+E). The exact space depends on the implementation, but if each edge uses 64 bits, the entire graph can be stored in about 4 million bits. This is much less than the 100 million bits used by an adjacency matrix.

If a graph has 10000 nodes and 50000 edges, and you want to be able to check as quickly as possible if a particular edge exists in the graph, how should you represent it?
A. Adjacency matrix
B. Array of adjacency lists
Adjacency matrix. This allows us to check for the existence of an edge in O(1) time.
