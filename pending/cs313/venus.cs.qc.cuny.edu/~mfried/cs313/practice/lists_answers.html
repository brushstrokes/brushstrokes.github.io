Write a method
List<Integer> even(List<Integer> input)
that returns a List of all even integers in the input List. 
List<Integer> even(List<Integer> input) {
   List<Integer> list = new ArrayList<>();
   for (int i = 0; i < input.size(); ++i)
      if (input.get(i) % 2 == 0)
         list.add(input.get(i));
   return list;
}
