Midterm advice

Some important topics for the midterm are listed below. The percentages are estimates and may not be accurate.
  • Know how to write Java code (35-40%).
    • You will have to write LinkedList methods, and methods that use sets or maps. You will have to write a method similar to the BST practice problems, or the add or contains method (but not the remove method). You will have to write a method using binary search.
  • Know which ADT or data structure to use in which situation (15-20%).
  • Know the runtimes of each data structure operation (10-15%).
  • Know how to draw a picture of a BST or AVL tree after adding or removing elements.
  • Be prepared to answer various other questions (most will be similar to the practice problems).
