Intro to data structures

In this course we will learn about different ways to store and organize data so that it can be used efficiently.

Abstract data types and implementations

An abstract data type (ADT) is a mathematical model of a data structure.
An ADT specifies the type of data stored and what operations can be performed.
A data structure is a concrete implementation of an ADT.

In Java, an ADT is represented by an interface with a list of method declarations (without an implementation).
A data structure is represented by a class, which can implement an interface (or interfaces) by defining every method in the interface(s).

Example
In Java, the list ADT is represented by the List interface.
Both the ArrayList class and the LinkedList class implement the List interface.

In this course we will learn about stacks, queues, trees, heaps, hash tables, and graphs.
Which one should you use? It depends what you're trying to do. Even if a program is correct, switching to a different data structure might make it run faster.

We will also learn some algorithms. An algorithm is a well-defined procedure that takes input and produces output.
